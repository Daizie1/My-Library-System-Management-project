📌 System Overview

The Mirambi Library System is a web-based application designed to help manage library activities efficiently. It allows users to add books, view available books, issue and return books, and keep track of the total number of books in the system.

🎯 Purpose of the System

The main purpose of this system is to simplify library operations by reducing manual record keeping. It improves accuracy, saves time, and provides an easy way for users to manage and access library resources.

⚙️ Key Features

  • Add new books to the system
  • View a list of all books
  • Issue and return books
  • Delete books from the system
  • Simple login authentication
  • Automatic book count display

💻 Technologies Used

This system was developed using the following technologies:

  • HTML – for structuring the web pages
  • CSS – for styling and layout design
  • JavaScript – for system functionality
  • LocalStorage – for storing data in the browser

⚠️ System Limitations

  • No backend database (data may be lost if browser storage is cleared)
  • Supports only one user at a time
  • Cannot be accessed online (offline system)

🚀 Future Improvements

  • Integration with a database (MySQL / Firebase)
  • Multi-user login system
  • Online access via internet
  • Book reservation and due date tracking
